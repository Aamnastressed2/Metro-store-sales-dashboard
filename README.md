
# 📊 Metro Store Sales Dashboard (Excel)

## 📌 Overview
This project presents an interactive Excel dashboard analyzing sales performance for a retail store in 2022. The goal was to clean raw data, transform it, and generate business insights through visualization.

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Data Cleaning Techniques

---

## 📊 Dashboard Features
- Sales vs Orders trend over time
- Order status distribution
- Customer demographics (Age & Gender)
- Top performing states
- Sales by channel

---

## 📈 Key Insights

- Sales peaked in early months and declined towards year end
- 92% of orders were successfully delivered
- Women contribute 64% of total sales
- Maharashtra is the top-performing state
- Amazon is the highest revenue-generating channel

---

## 💼 Business Impact
The dashboard helps identify key revenue drivers, customer segments, and operational performance, enabling better decision-making in marketing, logistics, and sales strategy.
